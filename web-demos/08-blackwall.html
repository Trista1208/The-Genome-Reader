The System

Three modules.
One defensive layer.

Strictly protective by construction — it predicts and explains resistance that already exists. It never designs, modifies, or suggests changes to an organism.

01 — Genome Reader

From DNA to features

A reconstructed FASTA is annotated with NCBI AMRFinderPlus, mapping known resistance genes and point mutations into a repeatable, documented feature space.

02 — Predictor

Will each antibiotic work?

One calibrated model per antibiotic, trained on laboratory-measured outcomes only — de-duplicated by genetic similarity so it learns resistance, not look-alikes.

03 — Decision Report

Know when not to predict

Every drug gets a verdict, calibrated confidence, and an evidence category. Weak or unfamiliar evidence returns a no-call — a strength, not a failure.
