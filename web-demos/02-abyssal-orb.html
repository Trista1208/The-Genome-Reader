The System

Genome Reader 01

A reconstructed FASTA becomes model features through NCBI AMRFinderPlus — known resistance genes and point mutations, mapped repeatably and documented end to end.

↗

Predictor 02

One calibrated model per antibiotic, trained only on laboratory-measured outcomes, de-duplicated by genetic similarity so it learns resistance — not look-alikes.

↗

Decision Report 03

Every drug returns a verdict, calibrated confidence, and an evidence category. Weak or unfamiliar evidence yields a no-call — by design.
