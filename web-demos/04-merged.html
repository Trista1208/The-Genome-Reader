The System

Three modules.
One defensive layer.

Strictly protective by construction — it predicts and explains resistance that already exists. It never designs, modifies, or suggests changes to an organism.

01_genome_reader

MODULE 01Genome Reader

A reconstructed FASTA is annotated with NCBI AMRFinderPlus — known resistance genes and point mutations mapped into a documented, repeatable feature space.

AMRFINDERPLUS
02_predictor

MODULE 02Predictor

One calibrated model per antibiotic, trained only on laboratory-measured outcomes — de-duplicated by genetic similarity so it learns resistance, not look-alikes.

GROUPED SPLITS
03_decision_report

MODULE 03Decision Report

Every drug gets a verdict, calibrated confidence, and an evidence category. Weak or unfamiliar evidence returns a no-call — a strength, not a failure.
